### Guide

#### 1. 圖鑑查詢

關鍵字支援以下幾種

- 編號
- 中日英名稱
- 特性/夢特
- 朱/紫
- 六星/6 星
- 努力值>0 的(Atk, Def, SpAtk, SpDef, Hp, Spd)
